# 持仓管理PRD

## 一、页面预期

可以查看我当前的基金持仓。

## 二、页面详情

页面主要由搜索栏和表格组成。

### 2.1、搜索栏

根据基金名称模糊搜索

根基基金代码搜索

### 2.2、表格字段

1. 基金代码
2. 基金名称
3. 当日收益（可排序）
4. 持有收益（可排序）

### 2.3、表格操作字段

1. 详情
   1. 点击详情按钮，展示详情页。详情页中，顶部是总览数据，纵览数据的字段为：
      1. 持有金额、持仓金额、持仓占比、持有收益、持有收益率、持仓成本、当日收益、持有收益、持有天数
   2. 下方是 基金的走势图
   3. 再下方是 基金重仓股
   4. 再下方是 行业分布
