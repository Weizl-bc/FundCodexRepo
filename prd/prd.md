# PRD

## 一、页面整体布局

顶部左侧为一级菜单栏，支持点级，点击后，asider menu的菜单会变化

顶部右侧是我的头像，点击可进行个人设置、登出等操作

中间左侧是asider menu，侧边栏菜单

中间除了左侧的菜单栏，就是内容框，在内容框的顶部有菜单栏面包屑，还有已打开的菜单，并且可以切换已打开的菜单。

## 二、菜单总览

菜单的主线：

数据中心 -> 市场总览 -> 我的基金 -> 预测与决策 -> 复盘分析 -> 策略/模型中心

### 2.1、菜单级别定义

顶部左侧是一级菜单

左侧asider menu是二级菜单，在二级菜单中，可以点击下拉，下拉内的菜单，输入三级菜单

### 2.2、菜单内容

#### 2.2.1、一级菜单

1. 市场总览
2. 我的基金
3. 预测与决策
4. 数据中心
5. 复盘分析
6. 系统设置
